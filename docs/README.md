# 📚 Документация cheburnet-router

Проект в процессе перехода со старой архитектуры (**v1**) на новую (**v2**). Поэтому
документация двухуровневая — здесь карта, что где и что читать первым.

> [!tip] Коротко
> - **v2 — куда идём.** Целевая архитектура, образовательная база знаний. **Начни отсюда.**
> - **v1 — что работает сегодня.** Документация текущей реализации (podkop/sing-box).
>
> v1 не «устаревшее», а **текущее, замещаемое v2**. Пока код v2 не заменит v1 —
> v1-главы остаются актуальными для живой системы.

## 🔀 Когда v1 станет архивом

Физический перенос v1-глав в `docs/v1/` (или `docs/legacy/`) произойдёт **синхронно с тем,
как код v2 заменит v1** — одним заходом, вместе с ретайром старого `README.md` и сворачиванием
`AGENTS.md` в `CLAUDE.md`. До этого момента v1-доки остаются на месте, чтобы не плодить битые
ссылки и не отрываться от живой системы. Удаления нет — есть архив + migration guide.
