# 📚 База знаний cheburnet-router v2

Это **Obsidian-vault** — документация проекта, устроенная так, чтобы быть одинаково
удобной **человеку** (граф взаимосвязанных заметок) и **ИИ** (чистый markdown как контекст).

> [!info] Образовательный проект в первую очередь
> cheburnet-router — учебный стенд. Цель документации не «как нажать кнопку», а
> **понять, как это работает от первых принципов**. Поэтому здесь нет чёрных ящиков:
> каждый сетевой механизм разобран до примитивов ядра.

## С чего начать

- 🏠 **[[Home]]** — карта всего vault'а (Map of Content). Главная точка навигации.
- 🧭 **[[conventions]]** — как устроен этот vault и как кормить его ИИ.
- 🎓 Новичку — иди по концепциям: [[split-routing]] → [[dnsmasq-nftset]] → [[policy-routing]].
- 🏗 Инженеру — [[data-plane]], [[engine-ucode]], [[reliability]].
- 🤔 «Почему так?» — решения в `decisions/`: [[0001-why-not-singbox]], [[0002-ucode-over-go]], [[0004-multi-protocol-tiers]].

## Как открыть

1. **Obsidian:** открой папку `docs/v2/` как vault (Open folder as vault). Заработают граф,
   backlinks, `[[wikilinks]]`.
2. **Просто GitHub/редактор:** это обычный markdown — читается и без Obsidian, ссылки кликабельны.
3. **Скормить ИИ:** см. [[conventions#Для ИИ]] — как собрать контекст одним куском.
